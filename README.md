
# آموزش قدر مطلق یک عدد اعشاری درسی شارپ

# نمونه کد C# برای محاسبه قدر مطلق

```csharp
        // عدد اعشا
        double number = -23.45;

        // محاسبه قدر مطلق
        double absoluteValue = Math.Abs(number);

        // نمایش نتیجه
        Console.WriteLine($"قدر مطلق {number} برابر است با: {absoluteValue}");
```
